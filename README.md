# cv

This is a simple readme file for my personal portfolio web page. 
I created this web page using HTML, CSS, and JS. The web page showcases some of my projects, skills, and achievements.
It also has a contact form where you can send me a message.

The web page consists of the following files:

- index.html: This is the main HTML file that contains the structure and content of the web page.
- style.css: This is the CSS file that defines the style and layout of the web page.
- script.js: This is the JS file that adds some interactivity and functionality to the web page.

To view the web page, you can open the index.html file in your browser. You can also modify the files to suit your preferences and needs. I hope you enjoy my portfolio web page and find it useful.
